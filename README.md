# Редактируемый список сотрудников

Тестовое задание для компании **[Ти Экс Консалтинг](https://krasnodar.hh.ru/employer/1559542?hhtmFrom=vacancy)**  
Позиция: **Junior Frontend-разработчик**

## Условия задания

Реализовать редактируемый список сотрудников.

Страница должна содержать:
- форму ввода / редактирования данных (в отдельном окне)
- список (таблицу) с уже введёнными сотрудниками с возможностью редактирования

Параметры сотрудника:
- имя
- фамилия
- стаж
- возраст
- адрес

Требования:
- использование Vue.js версии 3
- применение концепции MVC
- модульный подход к организации кода

---

## Стек

- Vue.js 3
- TypeScript
- Vite
- Composition API

## Реализованный функционал

- Отображение списка сотрудников в виде таблицы
- Добавление нового сотрудника
- Редактирование существующего сотрудника в модальном окне
- Валидация данных формы
- Переиспользуемые базовые UI-компоненты


## Архитектура

В проекте используется модульный подход и концепция MVC.

### Model
Хранит описание сущностей и типов данных, не зависит от UI и фреймворка:
- features/employee/model/

### View
Компоненты пользовательского интерфейса, отвечающие только за отображение и пользовательские события:
- features/employee/ui/
- features/shared/ui/

### Controller / Logic
Содержит бизнес-логику, валидацию и управление состоянием, изолированные от UI:
- features/employee/composables/
- features/employee/lib/

### Структура проекта
```
src/
├── features/
│ ├── employee/
│ │ ├── composables/
│ │ ├── lib/
│ │ ├── model/
│ │ └── ui/
│ └── shared/
│ └── ui/
├── pages/
├── styles/
├── App.vue
└── main.ts
```

## Установка и запуск

### Требования
- Node.js 24+

```bash
git clone git@github.com:npukojiucm/list-of-employees.git
cd list-of-employees
npm install
npm run dev
```

После запуска приложение будет доступно по адресу:
http://localhost:5173
